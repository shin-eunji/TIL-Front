### 리액트는 무엇인가

프론트엔드 라이브러리
DOM관리와 상태관리 업데이트
사용자 인터페이스와 기능개발 집중할 수 있도록!

> Angular: 다양한 기능이 내장되어있음. HTTP, 라우터, 다국어기능 등 내장되어 있습니다.
> Vue: webpack 모듈 번들러가 없어도 CDN으로 불러올 수 있습니다. 공식 라우터와 공식 상태 라이브러리가 있습니다. 디렉티브, 가상돔이 있습니다.
> React: 컴포넌트 개념!

### 리액트의 가상돔

MVC(model view controller)
양방향 바인딩
virtual DOM = 가상돔. 자바스크립트의 DOM

webpack: 코드들을 의존하는 순서대로 합쳐서 하나 또는 여러개의 파일로 결과물을 만들어냅니다. import을 하여 번들링 작업(빌드) 특정 확장자마다 처리하기 위한 준비하는 도구! 자바스크립트의 파일을 여러 개 만들었을 때 하나의 JS파일로 만들어줍니다.
babel: 자바스크립트 변환 도구! node.js, javascript 문법을 변환해줍니다.
